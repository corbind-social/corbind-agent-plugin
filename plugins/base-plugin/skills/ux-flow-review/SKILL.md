# UX Flow Review

## Instructions

1. Identify the primary user goal.
2. Evaluate entry, decision, and exit points.
3. Check error, empty, and edge states.
4. Flag friction and suggest concrete improvements.
5. Provide prioritized recommendations (high/medium/low).
